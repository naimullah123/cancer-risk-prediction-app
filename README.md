## Cancer Risk Prediction App

This is a Streamlit-based ML application that predicts cancer risk
using a Random Forest model trained on medical risk factors.

### Features
- Real-time prediction
- Pickled ML model
- User-friendly UI

### Tech Stack
- Python
- Scikit-learn
- Streamlit
